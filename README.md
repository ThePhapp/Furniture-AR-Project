## Luna Shop Furniture AR System

This project is a modern furniture e-commerce platform featuring Augmented Reality (AR) product viewing for Luna Shop.

### Key Features
- AR model viewer for furniture products (GLB/USDZ)
- Mobile-friendly AR experience (WebXR, ARCore, ARKit)
- Product details, categories, cart, and order management
- Admin dashboard and user authentication

### Technology Stack
- **Laravel** (PHP) for backend and API
- **Blade** for templating
- **MySQL** for database
- **Tailwind CSS** for UI styling
- **model-viewer** (Google) for AR rendering
- **JavaScript** for AR controls and UI logic
- **Docker** for deployment

### Quick Start
1. Clone the repository
2. Install dependencies (`composer install`, `npm install`)
3. Setup `.env` and database
4. Run migrations and seeders
5. Start the server (`php artisan serve`)
6. Access AR product pages to view furniture in 3D/AR
